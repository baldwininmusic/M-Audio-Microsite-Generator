1. Introduction

Thank you for purchase this product. Jet is a responsive megamenu component based in CSS and Javascript code.

2. Installation

Place the reference for CSS and Javascript files inside the “head” section of your page:

<link rel="stylesheet" type="text/css" href="jetmenu.css">

<script type="text/javascript" src="jquery-1.10.1.min.js"></script>

<script type="text/javascript" src="jetmenu.js"></script>

3. HTML markup

Jet is based in unordered lists. Example:

<ul class="jetmenu">

     <li class="active"><a href="#">Item 1</a></li>

     <li><a href="#">Item 2</a></li>

     <li><a href="#">Item 3</a></li>

     <li><a href="#">Item 4</a></li>

     <li><a href="#">Item 5</a></li>

</ul>

3.1 Right alignment

Just put the class "right" in the menu item:

<li class=”right”><a href="#">Item 2</a></li>

3.2 Dropdown menu

Just use a list with the class “dropdown”. See the example:

<ul class="jetmenu">

     <li class="active"><a href="#">Item 1</a></li>

     <li><a href="#">Item 2</a></li>

     <li><a href="#">Item 3</a></li>

     <li><a href="#">Item 4</a>

          <ul class="dropdown">

               <li><a href="#">Dropdown item</a></li>

               <li><a href="#">Dropdown item</a></li>

               <li><a href="#">Dropdown item</a></li>

               <li><a href="#">Dropdown item</a></li>

          </ul>

     </li>

     <li><a href="#">Item 5</a></li>

</ul>

3.2.1 Dropdown menu

The dropdowns can be aligned to right (default) or to left. To align the dropdowns to “left”, just use the class “left” on dropdown:

 <ul class="dropdown left">

      <li><a href="#">Dropdown item</a></li>

      <li><a href="#">Dropdown item</a></li>

      <li><a href="#">Dropdown item</a></li>

      <li><a href="#">Dropdown item</a></li>

</ul>

3.3 Megamenu

Just use a “div” with the class “megamenu”. See the example:

<ul class="jetmenu">

     <li class="active"><a href="#">Item 1</a></li>

     <li><a href="#">Item 2</a></li>

     <li><a href="#">Item 3</a></li>

     <li><a href="#">Item 4</a>

          <div class="megamenu">

          </div>

     </li>

     <li><a href="#">Item 5</a></li>

</ul>

After this, put your content inside the “megamenu” div.

4. Javascript configuration

Put the following code inside the “head” section of your page:

<script type="text/javascript">

     jQuery(document).ready(function() {

          $().jetmenu();

     });

</script>

4.1 Javascript options

Jetmenu has some option that you can choose:

Option

Type

Default value

Description

speed

integer (ms)

300

Submenu’s speed

indicator

boolean

true

Submenu’s indicator

hideClickOut

boolean

true

If a submenu is open, one click/touch out of the menu hides the submenu (landscape mode)

delay

integer (ms)

0

Defines the time delay before the submenu appears.

align

string

“left”

Menu alignment (“left” or “right”)

submenuTrigger

string

“hover”

Defines if the submenus appear after a mouse hover or a mouse click (“hover” or “click”)

Examples:

1) A menu with a submenu speed = 500 milliseconds:

<script type="text/javascript">

     jQuery(document).ready(function() {

          $().jetmenu({

               speed: 500

          });

     });

</script>

2) A menu without submenu’s indicators, speed = 600 milliseconds and mouse clicks:

<script type="text/javascript">

      jQuery(document).ready(function() {

           $().jetmenu({

                indicator: false,

                speed: 600,

                submenuTrigger: “click”

           });

      });

</script>

7. Useful informations and tips

a) Browsers compatibility

Jet has been tested on :

  • IE8+
  • Chrome
  • Firefox
  • Safari
  • Opera
  • Safari on Ipad/Iphone
  • Firefox/Chrome on Android

b) Real links

Is possible to use real links on dropdows parent items. It works this way:

     First tap/click: open dropdown

     Second tap/click: go to real link

c) Avoiding scroll up after a click

To avoid this, just replace fake links from “#” to “javascript:void(0)”.

d) Responsive behavior

Jet reacts (and go to mobile mode) if the screen width is minor than 768px (the most popular screen width on mobile devices).
